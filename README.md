##  Posts Generator

###  Installation

```bash
npm install
```

###  Démarrage

```bash
npm run dev
```

### ⚠ Ajoutez votre clé API Mistral dans le fichier .env ⚠️

###  Comment ça marche ?

#### 1- Choisissez le nombre de posts que vous souhaitez générer
#### 2- Choisissez les thèmes que vous souhaitez utiliser
#### 3- Générez vos posts Linkedin
